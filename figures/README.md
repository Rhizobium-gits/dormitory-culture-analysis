This directory contains the figures generated by the analysis scripts.

| `fig7_building_type_overview.png` | Building type overview | `analysis_main.R` |

## Generating Figures

To regenerate all figures, run the main analysis script:

```r
source("R/analysis_main.R")
```

Figures will be saved to this directory in PNG format at 300 DPI resolution.

## Note

Figure 1 (H village photographs) should be manually inserted and is not generated by the analysis code.
